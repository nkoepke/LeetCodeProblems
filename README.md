# LeetCodeProblems
A JSON representation of publicly available LeetCode Problems (not including premium problems) for usage in scientific research so you don't have to get all the problems by yourself

`code_problems.json`

```javascript
[
  {
        "id": string
        "title": string
        "difficulty": string
        "likes": number
        "dislikes": number
        "tags": array[string]
        "description": string
        "accessible": boolean
    },
    ...
]
```
